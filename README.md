# Dungeon-Master-Hub
  Resource application for dungeon masters to make their games and campaigns seamless. Providing resources to build campaigns, pull up a random encounter, or broaden a npc's character. 

  Building a campaign for an upcoming game night can be intimidating, especially when players have gone off the rails and are heading for disaster. Access a large database of D&D resources to find just the right random encounter to turn the story that much more fun. 
  Search tools to find monsters, npc’s, and spells for the next fight. A dictionary to access and build vocabulary for that one bard, pictures to express just what you want to show.

  Dungeon Master Hub aims to provide resources to help people create a smooth and exciting game. The biggest question for a dungeon master is “what’s next,” and not everyone excels at offering an answer. This application will provide basic information on creatures, spells, and resources to expand vocabulary for a more intelligent-sounding bard character, offer insight into the best spells to use, and allow the dungeon master to create a more in-depth experience.

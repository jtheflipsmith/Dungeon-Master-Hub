Introduction

Building a campaign for an upcoming games night can be intimidating, especially when players have gone off the rails and are heading for disaster. Access a large database of D&D resources to find just the right random encounter to turn the story that much more fun. Search tools to find monsters, npc’s, and spells for the next fight. Dictionary to build vocabulary for that one bard, pictures to express just what you want to show

Dungeon Master Resources aims to provide resources to help people create a smooth and exciting game. The biggest question for a dungeon master is “what’s next,” and not everyone excels at offering an answer. This application will provide basic information on creatures, spells, and resources to expand vocabulary for a more intelligent-sounding bard character, offer insight into the best spells to use, and allow the dungeon master to create a more in-depth experience.
